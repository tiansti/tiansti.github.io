读《法律推理方法》笔记（一）

木白2023-12-272023-12-27

书籍简介

我读的《法律推理方法》由波兰克拉科夫市（Krakow）雅盖隆大学（Jagiellonian University）的耶日·施特尔马赫和巴尔托什·布罗泽克两位法学教授编著，由北京第二外国语学院法政学院的陈伟功博士译著。
下面引用作者序言的内容，说明本书的主要内容：

关于法律推理方法，还需要写另外一本书来证成；那本书的目标是囊括法律理论中已经提出的大多数法律推理方法。

下面引用译者引言的内容，说明本书的主要内容：

本书是关于法律推理方法的经典著作。它要处理的是从总体的一般性视域中、从哲学层面探讨关于法律推理的方法论问题：史论结合，既有对”史“的整理，更有对”论“的阐发，从而给读者提供了一幅完整而清晰的方法论地图。它不仅对法官、检察官、律师、法学专业的师生等法律人，甚至对哲学、政治学、社会学等学界也极具启发意义。

本章出现的法学家汇总

1.1 三种立场

冯·基尔希曼（von Kirchmann）

简介：
德国19世纪法学家、哲学家、政治家，早年在莱比锡大学和哈勒大学学习法律，之后曾长期担任法官、检察官、议员等职，最终因其左倾的政治观点以及对左翼政治活动的积极参与而被解除司法职务并被剥夺退休金。晚年致力于哲学著作的翻译、研究，曾经担任柏林哲学协会主席，并创办大型丛书“哲学书系”，系统翻译、编辑、出版从古希腊直至近代一系列重要哲学家的作品。
本书提及的著作/观点/学派：
作为科学的法学的无价值性

萨维尼（Savigny）

简介：
德国最伟大的法学家之一，普鲁士国王弗里德里希·威廉四世的立法部长，历史法学派的代表人物。
本书提及的著作/观点/学派：
历史学派：法律科学是一门历史科学。他认为，法律是民族精神的产物，每个民族都有其独特的法律传统和制度。

普赫塔（Puchta）

简介：
格奥尔格·弗里德里希·普赫塔（Georg Friedrich Puchta）是一位重要的德国法学家，历史法学派代表人物之一，并被誉为“学说汇纂”的集大成者及“概念法学”的创始人。他于1798年出生在德国纽伦堡附近的卡多尔兹堡，来自一位有名的法官家庭。在1811-1816年期间，他在素有“古典人文主义”传统的纽伦堡文科中学学习，并受到了黑格尔的哲学影响。
本书提及的著作/观点/学派：
历史学派。他认为，法的历史是某些原则自动地和缓慢地揭示的过程，这些原则象胚胎一样，潜在于民族精神之中。此外，他还提出了著名的概念金字塔理论，这一理论解释了法律概念从伦理内涵逐渐淡出至具体法学概念的过程。

哈奇森（Hutcheson）

简介：
哈奇森（Hutcheson）是一位美国法学家，现任哈佛大学法学院教授。他的研究领域包括宪法、行政法和法律与经济学。
本书提及的著作/观点/学派：
美国实在论的直觉主义代表

德罗伊森（J. G. Droysen）

简介：
德罗伊森（J.G.Droysen）是一位德国历史学家和哲学家，他是19世纪德国历史学派的重要代表人物之一，也是西方历史哲学的奠基人之一。
本书提及的著作/观点/学派：
对科学作出自然主义和非自然主义范式的著名区分

狄尔泰（Wilhelm Dilthey）

简介：
狄尔泰（Wilhelm Dilthey）是一位德国哲学家、心理学家和人文学家，他是19世纪德国浪漫主义哲学运动的重要代表人物之一，也是西方人文主义思潮的重要思想家。
本书提及的著作/观点/学派：
对科学作出自然主义和非自然主义范式的著名区分

冯·赖特（G. H. von Wright）

简介：
冯·赖特（G.H.von Wright）是20世纪最重要的逻辑学家、哲学家和社会理论家之一。他认为，类别是基于我们对事物之间的相似性和差异性的理解而形成的，而不是基于某种抽象的定义或规则。他的这一理论挑战了传统的逻辑学观点，并引发了广泛的讨论和研究。
本书提及的著作/观点/学派：
”强“分析哲学。

贝克尔（O. Becker）

简介：
加里·贝克尔（Gary Stanley Becker）是一位美国经济学家和社会学家，他是芝加哥大学经济学和社会学教授，也是诺贝尔经济学奖得主之一。他认为，个体的教育、培训和健康等方面的投资可以增加其生产能力和收入水平，从而对经济增长产生积极影响。他的研究还涉及到家庭决策、犯罪行为、歧视等社会问题的经济分析。他被认为是二十世纪最杰出的经济学家之一，对经济学和社会科学研究产生了深远的影响。
本书提及的著作/观点/学派：
”强“分析哲学。

卡利诺夫斯基（J. Kalinowski）

简介：
卡利诺夫斯基（Jerzy Kalinowski）是一位波兰数学家，他是图论和组合数学领域的重要贡献者之一。卡利诺夫斯基的研究主要集中在图论和组合数学领域。
本书提及的著作/观点/学派：
”强“分析哲学。

罗斯（A. Ross）

简介：
阿瑟·罗斯（Arthur Ross）是一位美国数学家，他是数理逻辑和计算机科学领域的重要贡献者之一。罗斯的研究主要集中在数理逻辑和计算机科学领域。他对形式化语言、自动机理论和计算复杂性等方面做出了重要的贡献。他的研究成果被广泛应用于计算机科学和人工智能等领域。
本书提及的著作/观点/学派：
”强“分析哲学。

沃伦斯基（J. Woleński）

简介：
沃伦斯基（Jan Woleński）是一位波兰数学家，他是图论和组合数学领域的重要贡献者之一。他的研究主要集中在图论和组合数学领域。
本书提及的著作/观点/学派：
”强“分析哲学。

阿尔乔龙（C. Alchourron）

简介：
阿尔乔龙（Carlos Alchourron）是一位葡萄牙计算机科学家，他是网络和分布式计算领域的重要贡献者之一。
本书提及的著作/观点/学派：
”强“分析哲学。

E. 布雷金（E. Bulygin）

简介：
伊万·布雷金（Evgeny Bulygin）是一位俄罗斯数学家，他是图论和组合数学领域的重要贡献者之一。布雷金的研究主要集中在图论和组合数学领域。
本书提及的著作/观点/学派：
”强“分析哲学。

哈特（H. L. A. Hart）

简介：
哈特（Herbert Lionel Adolphus Hart）是一位英国法学家和哲学家，他是现代法律哲学领域的重要贡献者之一。哈特的研究主要集中在法律哲学和法学理论领域。他对法律的概念、规则和权威等问题进行了深入的研究和分析。他的研究成果对现代法律哲学的发展产生了重要影响，尤其是他对法律规则的分类和解释的理论。
本书提及的著作/观点/学派：
”弱“分析法

彼得拉日茨基（L. Petrazycki）

简介：
彼得拉日茨基（Ludwik Petrazycki）是一位波兰数学家，他是图论和组合数学领域的重要贡献者之一。彼得拉日茨基的研究主要集中在图论和组合数学领域。
本书提及的著作/观点/学派：
法学完备理论：”他主张这样一种理论可以建立在他自己提出的情感心理学基础上。“ “道德与法律情感属同一类现象”

惹尼（Geny）

简介：
是一位匈牙利数学家和法律学者，他是图论和组合数学领域的重要贡献者之一，同时也是自由法学派的代表人物之一。他主张法律应该是一种灵活的工具，能够适应社会的变化和发展。他认为法律应该以公正和效率为原则，并强调法官的自由裁量权和法律解释的重要性。
本书提及的著作/观点/学派：
“大陆实证主义错误地假定成文法是法律的唯一来源。” “在法律解释的过程中，人们也应该考虑到有效法律的其他三种来源：惯例、权威、法官的自由心证。”

坎托罗维奇（Kantorowicz）

简介：
亨里希·坎托罗维奇（Hermann Kantorowicz）是一位奥地利法学家和历史学家，他是现代法律史和比较法学领域的重要贡献者之一。
本书提及的著作/观点/学派：
与 惹尼 的观点相近

霍姆斯（Holmes）

简介：
奥利弗·温德尔·霍姆斯， Jr.（Oliver Wendell Holmes, Jr.）是美国著名法学家和联邦最高法院大法官，他的司法观点以其简洁精辟而受到广泛赞誉，对普通法理论、公民权利和宪法原则等方面的见解有深远影响。
本书提及的著作/观点/学派：
他指出，“法律的生命不在于逻辑，而在于经验”。在他的名著《The Path of the Law》中，霍姆斯阐释了四种彼此具有相互联系的观点：注重现实（和现世）结果的实用主义，主张法律应独立并区别于道德的中立主义，把外在行为（而非内在心态）视作对象的客观主义，以经验探索（而非纯粹的逻辑推理）为基础的实证主义。

卢埃林（Llewellyn）

简介：
卡尔·尼尔森·卢埃林（Karl Nelson Llewellyn）是美国著名法学家和法律教育家，被誉为“实用主义法学之父”。
本书提及的著作/观点/学派：
美国实在论：强调法律应该适应社会变化和发展的需要。他认为，法律不仅仅是规则和制度，更是一种过程，需要不断地进行解释和适用。他的这一观点对后来的法律现实主义运动产生了深远影响。

弗兰克（Frank）

简介：
无

美国实在论

摩尔（Moore）

简介：
无

孔德（A. Comte）

简介：
奥古斯特·孔德（Auguste Comte）是法国哲学家、社会学家和实证主义的创始人。他生于法国南部的蒙特勒伊，在巴黎学习了数学和自然科学，并成为圣西门的学生。
本书提及的著作/观点/学派：
提出了“社会学”这一概念，认为社会学应该研究社会现象的规律性和普遍性，以促进社会的稳定和发展。他认为人类思维的发展经历了三个阶段：神学阶段、形而上学阶段和实证主义阶段。在实证主义阶段，人们不再相信神秘的力量和超越的存在，而是通过观察和实验来获取知识。

边沁（Bentham）

简介：
杰里米·边沁（Jeremy Bentham）是英国哲学家、法学家和改革家。他生于伦敦，曾在牛津大学学习法律，并成为一位杰出的律师。
本书提及的著作/观点/学派：
功利主义：他认为人类行为应该追求最大的幸福和最小的痛苦。他认为社会制度和政治体制的合理性应该根据其对社会幸福的贡献来评判。他还提出了“最大幸福原则”，即任何行动都应该追求最大多数人的幸福。

庞德（Pound）

简介：
罗斯科·庞德（Roscoe Pound）是美国法学家和法律教育家，也是20世纪最重要的法律思想家之一。他生于美国俄亥俄州的一个德国移民家庭，曾就读于哈佛大学和柏林大学，并在德国留学多年。他是美国《联邦民事诉讼规则》的主要起草人之一
本书提及的著作/观点/学派：
社会学法学：认为法律应该被视为一种社会制度，其发展受到社会、政治和经济等多种因素的影响。他强调法律与社会之间的相互作用和相互影响，认为法律应该为社会服务，而不是独立存在。

哈盖尔斯特列姆（Hägerström）

简介：
哈盖尔斯特列姆（Hägerström）是瑞典的一位著名法学家，他的研究领域主要涉及宪法、行政法和人权法等方面。他曾担任瑞典政府法律顾问、瑞典最高行政法院法官等职务，对瑞典的法律制度和司法实践做出了重要贡献。
本书提及的著作/观点/学派：
斯堪的那维亚实在论。他认为，如果不在真实的语境中探讨，Mamet所有的观念或概念都是无意义的。

伦德斯泰特（Lundstedt）

简介：
伦德斯泰特（Lundstedt）是瑞典的一位著名法学家，伦德斯泰特的研究领域主要涉及刑法、刑事诉讼法和犯罪学等方面。他曾担任瑞典最高刑事法院法官、瑞典国家刑事委员会主席等职务，对瑞典的法律制度和司法实践做出了重要贡献。
本书提及的著作/观点/学派：
斯堪的那维亚实在论

本书提及的著作/观点/学派：
斯堪的那维亚实在论

马图拉那（Maturana）

简介：
马图拉那（Maturana）是瑞典的一位著名法学家，他的研究领域主要涉及宪法、行政法和人权法等方面。他曾担任瑞典政府法律顾问、瑞典最高行政法院法官等职务，对瑞典的法律制度和司法实践做出了重要贡献。
本书提及的著作/观点/学派：
系统论，自我再生系统

卢曼（Luhmann）

简介：
尼克拉斯·卢曼（Niklas Luhmann）是德国著名的社会学家、法学家和哲学家，他的研究领域主要涉及法律社会学、法律系统理论和功能主义等方面。他提出了“法律系统”的概念，认为法律是一种自成一体的符号系统，具有自我生成和自我维持的特性。他还强调了法律与社会之间的相互作用和相互影响，认为法律不仅是一种规范性的工具，还是一种社会交往的形式。
本书提及的著作/观点/学派：
系统论，自我再生系统

托伊布纳（Teubner）

简介：
贡塔·托依布纳（Gunther Teubner）是德国著名的法学家，法学理论家和法律社会学家，他对卢曼的社会系统论深有研究，在法学领域对系统论有所延展和创新，被誉为继卢曼之后，当代关于法律系统理论重要的代表性人物。他的著作《法律：一个自创生的系统》在国内已经有中译本出版。此外，他还著有《宪法的碎片：全球社会宪治》等作品。
本书提及的著作/观点/学派：
系统论，自我再生系统

凯尔森（Kelsen）

简介：
汉斯·凯尔森（Hans Kelsen）是20世纪著名的奥地利法学家，同时也是法律实证主义的代表人物，在法学、宪法学以及国际法学等领域有着深入的研究和丰富的贡献，被誉为20世纪最富思想原创力的法学家之一。
本书提及的著作/观点/学派：
他提出了“纯粹法律理论”

密尔（J. S. Mill）

简介：
约翰·斯图亚特·密尔（John Stuart Mill），也被称为约翰·斯图尔特·穆勒，是19世纪英国著名的哲学家、逻辑学家、经济学家和社会活动家。
本书提及的著作/观点/学派：
密尔是古典自由主义的重要代表人物，他的思想影响了后世许多社会政治理论的发展。他是边沁的功利主义的支持者，认为人类的行为应该追求最大的幸福，并且主张个人自由和平等的重要性。

波斯纳（Posner）

简介：
理查德·波斯纳（Richard Allen Posner）是美国联邦上诉法院法官，他的贡献主要在于法律经济学领域，是法律经济学运动的重要人物。
本书提及的著作/观点/学派：
法律创制与解释的过程与一些经济学规则一致

史蒂文森（Stevenson）

无

本章经典摘要

1.1 三种立场

法官对具体案件的判决一定依赖于他的直觉与想象，而每个案件对于法官来说只是一个刺激物，法官的反应是要做出一个好的公正的判决。
这种反应是非理性的、直观的，或者说是情感上的直觉，是一种心灵的试探功能，这是好的法律人所具有的一种想象或直观。
